Dado um número inteiro positivo n, encontre a soma de todos os números inteiros no intervalo [1, n] inclusive que são divisíveis por 3, 5ou 7.

Retorna um número inteiro denotando a soma de todos os números no intervalo fornecido que satisfaz a restrição.

 

Exemplo 1:

Entrada: n = 7
 Saída: 21
 Explicação: Números no intervalo [1, 7]que são divisíveis 3por 5,ou 7 são 3, 5, 6, 7. A soma desses números é 21.
Exemplo 2:

Entrada: n = 10
 Saída: 40
 Explicação: Números no intervalo [1, 10] that aredivisíveis 3por 5,ou 7são 3, 5, 6, 7, 9, 10. A soma desses números é 40.
Exemplo 3:

Entrada: n = 9
 Saída: 30
 Explicação: Números no intervalo [1, 9]que são divisíveis por 3, 5ou 7são 3, 5, 6, 7, 9. A soma desses números é 30.
 

Restrições:

1 <= n <= 103
